# 马上 2015

- pubdate: 2014-01-27

---

回望去年和前年的寥寥数篇文章就觉得自己太懒了，懒并不只表现在写博上，去年的平均起床时间已经比以往晚了一个小时以上，更有继续推迟的趋势，某人已经放弃治疗。

再回望去年其实还有不少新鲜事。

感觉自己快要从一个前端工程师变成一个 Node 工程师了，Node 技能掌握不少，github 上满屏的绿点带来自我满足，但被关注的太少，也许明年写总结的时候会变成后端工程师。今年在 github 上比较活跃，慢慢习惯了混社区和他们交流，不管你英文多搓，老外都会非常耐心的回答问题，对学习英文也比较有帮助。

除此之外，学英语的方法还有出国。今年连续出了两次国（[马来](http://chanyouji.com/trips/69053)和台湾），如果你不出去你将永远不知道外面的世界，用自己的眼睛去看，用自己的耳朵去听，用自己的嘴去尝，用自己的脚去走。我们在异国他乡坐错车到了计划外距离200km的城市，在海岸和机场的中间飙着车，抓住每个傍晚的机会去追逐日落。

行万里路也要读万卷书，小时候不用功只能老了补，最近在[整理买过的书](http://book.douban.com/doulist/3462026/)后发现基本都是与日本相关的，而且对历史越发感兴趣，除此之外还看了不少轻小说。

这一年公司有了不小调整，将近5年的工作时间中团队已经完全换血了，现在只能靠我们几个老骨头挖坟了，但绝不倚老卖老。让自己不再内向，让自己有话直说，指出领导问题称赞新人的表现。

身边的人慢慢的都结婚了、有了孩子、孩子打酱油，我也该领证了。

2014 已到，马上 2015。
